## 🎨 自定义颜色

修改 `:root` CSS 变量：

```css
:root {
    --accent: #00ff88;           /* 主强调色 - 霓虹绿 */
    --accent-secondary: #00ccff; /* 次强调色 - 霓虹蓝 */
    --orange: #fd6f32;           /* 警告/难点标记 */
}
```
